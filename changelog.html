Release Notes & Changelog

Stay up to date with the latest releases, performance improvements, and fixes for Nepali DatePicker Studio.

v1.2.0 June 27, 2026 (Latest Release)

Assets Separation, Theme Performance & Dynamic Booking Initialization

  • FeatureClean Separation of CSS & JS Assets: Extracted all page-specific inline stylesheets and script logics from all HTML files into modular external assets (e.g. assets/css/playground.css, assets/js/playground.js, etc.) for streamlined maintenance and browser caching.
  • FeatureEarly Theme Initializer Script: Unified the head theme-detection code into assets/js/ndp-theme-init.js inside all HTML headers to immediately apply the light/dark themes and avoid layout shifts.
  • FeatureStandalone Age & Date Difference Calculator: Added an interactive conversion tab allowing users to calculate precise age, total elapsed days, and relative time offsets dynamically between two Bikram Sambat dates.
  • FeatureLong AD Gregorian Format (DD MMMM YYYY): Shipped built-in format conversion allowing high-accuracy mapping of Nepali BS dates to standard long-form English AD dates (e.g., 01 February 2035).
  • FeatureBilingual Devanagari Translation Cleanups: Replaced literal Devanagari translation terms like सफा (clean) with the action-oriented term हटाउनुहोस् (clear/remove) in calendar panel layouts.
  • FixBooking & Reservation Pickers Fix: Resolved a syntax brackets error within the playground scripts and successfully initialized all 7 dynamic booking form styles, matching travel, medical, and dining formats.
  • FixUTF-8 Encoding Normalization: Swapped custom unicode box-drawing character strings (e.g., ──) causing decoding warnings in specific server setups with standard CSS formatting rules.
  • FixSelf-Defense Security Seals: Added prototype freeze validations and DOM tree MutationObservers to secure calendar widgets from script-injection or hot-patching.
  • FixRefreshed ZIP Archive Distribution: Re-compiled the main download package (nepali-datepicker-pro.zip) to distribute clean, modular, and lightweight code files alongside the new asset structures.
v1.1.0 June 2026 (Current Release)

Performance Tuning, Time Picker Integration & Layout Fixes

  • FeatureLight / Dark Theme Switcher: Built a client-side theme switcher utilizing CSS custom property variables and localStorage to persist preference states across page reloads.
  • FeatureRange Presets Sidebar: Integrated quick date range presets for single-click selection of common intervals, including Nepalese Fiscal Year (आ.व.) ranges.
  • FeatureDual Time Picker: Embedded hour and minute selector dropdown inputs in the calendar panel footer to allow simultaneous date and time capture.
  • FeatureWeekday & Boundary Constraints: Enabled disabling specific days of the week (e.g. Saturdays) and setting hard `minDate` and `maxDate` boundaries.
  • FixMemory Leak Cleanup: Refactored month navigation rendering to reuse a single document fragment instead of continuously recreating calendar container nodes.
  • FixStray Character Input Masking: Handled manual keypress typing events to automatically block invalid alphabetic characters and enforce formatting rules.
  • FixZ-Index Modals Stacking: Resolved overlay conflict bugs in relative layouts by absolute-positioning the picker panel dynamically on viewport focus.
  • FixAuto-Reset Range Selection Bounds: Fixed range selection limits so that selecting a start date chronologically after an end date auto-resets the endpoint instead of breaking calculations.
v1.0.0 May 2026

Initial Release

  • FeatureCore BS-AD Converter Engine: Mapped high-precision mathematical calendar mapping lookup matrices for Bikram Sambat years 1970 to 2100.
  • Feature22 Pre-Designed CSS Themes: Shipped built-in styling sheets (including midnight, glassmorphism, corporate) configured fully via custom CSS properties.
  • FeatureVanilla JS Footprint: Designed the widget as a lightweight, zero-dependency drop-in library to ensure framework compatibility.
  • FeatureFocus & Keyboard Controls: Enabled complete navigation of month/year panels and day cells using arrow keys, Esc, and Enter.
